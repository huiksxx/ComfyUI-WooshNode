# ComfyUI-WooshNode
Sony Woosh 音频生成自定义节点 | 文本/视频生成音频

## 功能
- 文本生成音频
- 短视频生成音频
- 长视频分段生成音频

## 安装
1. 打开 ComfyUI/custom_nodes 文件夹
2. 克隆仓库：
git clone https://github.com/huiksxx/ComfyUI-WooshNode.git
3. 重启 ComfyUI

## 模型放置
将 Woosh 官方模型放入：ComfyUI/models/woosh/checkpoints/
synchformer 权重放入：ComfyUI/models/
